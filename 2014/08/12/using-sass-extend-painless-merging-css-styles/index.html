究竟Mixin與extend的使用時機該如何抉擇？

基本上呢，

@mixin是將程式碼帶入到對應的class去，同時可帶入變數。

@extend則是藉由合併class，並吃到共通樣式，但沒辦法帶入變數。

所以如果你的樣式都固定不變的，不會需要帶參數去改變樣式的話，

使用@extend，程式碼會比較少些。

但如果你的程式碼需要帶入多個變數進行運算時，那用@mixin則較適合。
